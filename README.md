**Learning Outcomes**


1️⃣ Understand REST Architecture

Explain REST principles (Client–Server, Stateless, CRUD operations)

Understand HTTP methods: GET, POST, PUT, DELETE

Work with HTTP status codes (200, 201, 404, 415, etc.)

2️⃣ Develop Backend Using Flask

Install and configure Flask

Create a backend server in Python

Define routes using @app.route

Handle JSON requests and responses

3️⃣ Implement CRUD Operations

Create (POST) new student data

Read (GET) all students and single student

Update (PUT) student details

Delete (DELETE) student records

Store and manage data using in-memory array

4️⃣ Test APIs Using Postman

Send HTTP requests using Postman

Set request headers (Content-Type: application/json)

Send JSON request body

Verify responses and status codes

5️⃣ Version Control Using Git & GitHub

Initialize Git repository

Commit and push code to GitHub

Manage branches and resolve push errors

6️⃣ Deploy Application on Cloud

Deploy Flask app on Render

Configure build & start commands

Use gunicorn for production deployment

Generate public demo link

7️⃣ Documentation Skills

Create README.md
